# 개인정보처리방침 — ShiftSleep

최종 업데이트: 2026-04-08

ShiftSleep("본 앱")은 사용자의 개인정보를 소중히 다룹니다. 본 방침은 본 앱이 수집·이용·보관하는 정보와 사용자의 권리를 설명합니다.

## 1. 수집하는 정보

### 1-1. 사용자가 직접 입력하는 정보 (로컬 저장)
- 직군 / 시프트 종류 / 근무표 / 사이클 시작일
- 수면 기록 (취침·기상 시각, 메모, 수면 품질 등)
- 알림 설정 (알림 시간 등)

위 정보는 모두 **사용자 기기 내부**에만 저장되며, ShiftSleep 서버 또는 제3자에게 전송되지 않습니다.

### 1-2. 자동 수집 정보
- **광고 ID (AAID)**: Google AdMob이 광고 게재 및 측정을 위해 사용
- **광고 로그**: AdMob SDK가 광고 노출/클릭 이벤트를 Google에 전송
- **크래시 로그** (선택): Firebase Crashlytics 사용 시 익명 크래시 데이터

## 2. 정보 이용 목적

| 정보 | 목적 |
|---|---|
| 시프트/수면 기록 | 룰 기반 수면 추천, 통계 표시 |
| 광고 ID | 맞춤형 광고 게재 |
| 크래시 로그 | 앱 안정성 개선 |

## 3. 정보 공유

본 앱은 사용자가 직접 입력한 정보를 외부에 공유하지 않습니다.
다음 제3자 SDK가 광고/안정성 기능을 제공하며, 각각의 개인정보처리방침이 적용됩니다.

- **Google AdMob**: https://policies.google.com/privacy
- **Firebase**: https://firebase.google.com/support/privacy

## 4. 보관 기간

- 사용자 입력 정보: 사용자가 앱을 삭제할 때까지 기기에 보관
- 광고/크래시 데이터: 각 SDK 정책에 따름

## 5. 사용자 권리

- 앱 내 모든 데이터는 시스템 설정 → 앱 → ShiftSleep → 저장공간 비우기로 즉시 삭제 가능
- Android 시스템 설정에서 광고 ID 재설정 또는 광고 맞춤설정 거부 가능
- Android 13 이상에서는 시스템 설정에서 알림 권한 철회 가능

## 6. 만 14세 미만 아동

본 앱은 교대근무자를 대상으로 하며 만 14세 미만 아동의 사용을 의도하지 않습니다.

## 7. 방침 변경

방침이 변경되면 본 페이지의 "최종 업데이트" 일자를 갱신합니다. 중요한 변경은 앱 내 공지로 안내합니다.

# Privacy Policy — ShiftSleep (English)

Last updated: 2026-04-08

ShiftSleep ("the App") respects your privacy. This policy explains what data the App collects, how it is used, and your rights.

## 1. Information We Collect

### 1-1. User-entered (stored locally)
- Occupation, shift types, work schedule, cycle start date
- Sleep records (start/end times, notes, quality)
- Notification preferences

All of the above is stored **only on your device** and never transmitted to our servers or third parties.

### 1-2. Automatically collected
- **Advertising ID (AAID)**: used by Google AdMob for ad delivery and measurement
- **Ad event logs**: AdMob SDK sends impression/click events to Google
- **Crash logs** (optional): anonymous crash data via Firebase Crashlytics

## 2. Purpose of Use

Personal entries → in-app sleep recommendations and statistics
Advertising ID → personalized ads
Crash logs → app stability

## 3. Sharing

We do not share user entries with any third party. The following SDKs operate under their own privacy policies:

- Google AdMob — https://policies.google.com/privacy
- Firebase — https://firebase.google.com/support/privacy

## 4. Retention

Locally entered data persists until you uninstall the app or clear its storage. Third-party SDK data follows the respective providers' retention policies.

## 5. Your Rights

- Delete all local data via system Settings → Apps → ShiftSleep → Clear storage
- Reset Advertising ID or opt out of personalization in Android system settings
- Revoke notification permission in system settings (Android 13+)

## 6. Children Under 14

The App targets shift workers and is not intended for children under 14.

## 7. Changes

When this policy changes the "Last updated" date will be revised. Material changes will be announced inside the App.
